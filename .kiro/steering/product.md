# Product Overview

Real Estate Data API and Web Application - a full-stack cloud-native MVP for managing real estate properties, users, and realtors.

## Purpose

Academic project demonstrating core full-stack development and cloud deployment skills. Built as a Minimal Viable Product (MVP) to showcase:

- RESTful API development with C# ASP.NET Core
- React frontend consuming the API
- AWS cloud deployment (RDS, ECS Fargate, S3)
- API gateway integration (Google APigee)

## Core Entities

- **Properties**: Real estate listings with address, price, bedrooms, bathrooms, square footage, type, status, and realtor association
- **Users**: End users with contact information and account status
- **Realtors**: Licensed agents managing property listings with experience and specialization details

## Key Features

- Complete CRUD operations for all three entities
- RESTful API with Swagger/OpenAPI documentation
- Repository pattern for clean data access
- DTO-based API contracts with AutoMapper
- React web client for data visualization and management
- Cloud deployment on AWS infrastructure
- API security and access control via APigee gateway

## Target Users

- Academic assessors evaluating full-stack development competency
- Development team gaining practical cloud deployment experience
- Students learning API consumption and integration patterns
